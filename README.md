# Material-HTML-CSS
Materiais do mini-curso de HTML e CSS desenvolvido na disciplina de Informática e Sociedade da UEM
___
## Aula 1:
  - Estrutura fundamental de um documento HTML
  - Tags de título (`<h>`)
  - Tag parágrafo e quebra de linha (`<p>` e `<br>`)
___
## Aula 2:
  - Inserção de imagens (`<img>`)
  - Listas (`<ul>`,`<ol>`,`<li>`)
  - Entrada de dados (`<form>`,`<input>`)
  - Extensão LivePreview
___
## Aula 3:
  - Inserção de links com a tag âncora (`<a>`)
  - Criação de vínculo com URL externo
  - Criação de páginas para cada receita
  - Vínculo com outro arquivo HTML
  - Criação de uma página principal de exibição de receitas
__
## Aula 4:
  - Introdução ao CSS
  - Tipos de estilização (inline, interna e externa)
  - Tipos de seletores básicos (elementos, classes e identificadores)
___
## Aula 5:
  - Estilização da tela inicial
___
## Aula 6:
  - Estilização das telas das receitas
  - Finalização do projeto
___
## Aula 7:
  - Finalização do curso
  - Correções e pequenas modificações finais
